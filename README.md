
## SDK 简介

本SDK是在神策开源的可视化埋点SDK上进行二次开发。

## 技术栈
### 1、Android studio 插件开发
### 2、Transform的相关API
### 3、字节码插桩及ASM使用
### 3、ClassLoader、gradle、DSL语言groovy

## 基本要求
Android 4.0 及以上。
